Projetos com sites

Trabalhei na parte de arquitetura e criação de CSS nos seguintes sites abaixo,(durante os projetos dos cursos da Alura).No decorrer de cada projeto acabei desenvolvendo algumas habilidades com CSS, visando a organização do código, também utilizando da parte do Flexbox e CSS Grid para a distribuição dos elementos dentro de cada projeto estudado.

O projeto "Fruta & fruto foi desenvolvido com o intuito de trabalhar a organização do código CSS, com o intuito de quebrar o código em partes, com cada sessão tendo seu próprio style, além da utilização adequada do Class entre elementos pais e filhos."

O projeto do site Alura teve intuito de recriar a estrutura do site da Alura, utilizando especificamente o Flexbox como base para a distribuição dos elementos. Já o projeto Alura Store visou a criação de uma plataforma de aplicativos, utilisando o CSS grid para distribuir os elementos da página.
